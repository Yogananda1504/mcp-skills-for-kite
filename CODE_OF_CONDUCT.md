# Code of Conduct

This repository aims to be welcoming, respectful, and professional.

## Expected behavior

- Be respectful in issues, pull requests, and discussions.
- Focus feedback on the work, not the person.
- Use constructive language and assume good intent.
- Keep discussions on-topic and productive.

## Unacceptable behavior

- Harassment, abuse, or discriminatory language.
- Personal attacks, trolling, or intimidation.
- Spam, deliberate disruption, or repeated bad-faith behavior.

## Enforcement

Project maintainers may remove or reject contributions that violate this code of conduct.
Repeated or severe violations may result in being blocked from future participation.

## Reporting issues

If you need to report a conduct issue, open a private security-style contact channel in your GitHub project settings or contact the repository maintainers directly.
